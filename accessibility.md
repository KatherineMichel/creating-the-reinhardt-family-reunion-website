# Accessibility

## Color Scheme

The website color scheme passes the [WebAim Color Contrast Checker](http://webaim.org/resources/contrastchecker) with a contrast ratio of 17.09:1.

* Background color: #1a1c1e
* Foreground color: #ffffff
* Accent color: #0aac8e
